# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Aravbgp2006/blood-donation-project.git
```

Go to project

```bash
cd blood-donation-project
```

Install packages

```bash
npm install
```

Run project

```bash
npm run dev
```
